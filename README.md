# 🎓 AI Study Planner — Smart AI Learning & Schedule Assistant

An intelligent, full-stack web application designed to help students supercharge their learning journey. Features automated syllabus PDF parsing, AI schedule generation, interactive Pomodoro timers, exam countdowns, subject difficulty tracking, and AI tutoring powered by Groq LLM.

## 🚀 Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Lucide Icons, React Router 7
- **Backend**: Node.js, Express 5, MongoDB, Passport.js (Google OAuth 2.0), Groq SDK
- **Deployment**: Vercel (Frontend), Render (Backend)
